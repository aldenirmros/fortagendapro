Como funciona a integracao
  • O sistema acessa sua Google Agenda via API do Apps Script
  • Os eventos sao importados com cor vermelha para diferenciar
  • A sincronizacao busca eventos da semana atual visualizada
  • Cada usuario sincroniza apenas sua propria agenda Google
  • Para funcionar, o Web App precisa de permissao de Calendar
Use o e-mail da conta Google (nao cole a URL, apenas o e-mail). Pode colar a URL completa que o sistema extrai automaticamente.
